# Modul 3 - CSS Fundamentals

## Overview

Modul ini membahas dasar-dasar CSS untuk styling dan layout halaman web. Siswa akan belajar tentang selector, box model, typography, layout dengan Flexbox & Grid, dan responsive design.

## Struktur Folder

- `1. Starter/` - Code awal untuk memulai praktik
- `2. Finished/` - Code final hasil praktik lengkap

## Learning Objectives

Setelah menyelesaikan modul ini, siswa akan mampu:

1. Memahami selector, class, dan id
2. Menguasai box model (margin, padding, border)
3. Mengatur typography, warna, dan background
4. Menggunakan Flexbox dan Grid untuk layout
5. Membuat responsive design dengan media query
6. Mendesain halaman profil yang menarik dan responsif

## Project Output

**Mini Project: Halaman Profil yang Menarik & Responsif**

- Styling modern dengan CSS
- Layout responsive menggunakan Flexbox/Grid
- Typography yang menarik
- Color scheme yang konsisten
- Mobile-first responsive design
