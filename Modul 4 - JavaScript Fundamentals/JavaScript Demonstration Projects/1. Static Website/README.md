# Static Website - Tanpa JavaScript

## Deskripsi

Website ini dibuat hanya dengan HTML dan CSS, tanpa JavaScript. Demonstrasi ini menunjukkan bagaimana website terlihat dan berperilaku tanpa JavaScript.

## Fitur yang TIDAK Tersedia

- Interaksi pengguna (klik, hover, input)
- Konten dinamis
- Notifikasi atau popup
- Validasi form real-time
- Animasi interaktif
- Respons terhadap aksi pengguna

## Struktur File

```
1. Static Website/
├── index.html          # Halaman utama
├── styles.css          # Styling CSS
└── README.md          # Dokumentasi ini
```

## Cara Menjalankan

1. Buka file `index.html` di browser
2. Coba klik tombol-tombol (tidak akan terjadi apa-apa)
3. Coba isi form (tidak ada validasi)
4. Perhatikan bahwa website terlihat statis

## Tujuan Pembelajaran

- Memahami keterbatasan website tanpa JavaScript
- Melihat perbedaan antara website statis dan interaktif
- Menyadari pentingnya JavaScript dalam web development modern

## Perbandingan dengan Website Interaktif

| Aspek     | Website Ini (Statis) | Website Interaktif    |
| --------- | -------------------- | --------------------- |
| Tombol    | Tidak berfungsi      | Berfungsi dengan baik |
| Form      | Tidak ada validasi   | Validasi real-time    |
| Animasi   | Hanya CSS            | JavaScript + CSS      |
| Feedback  | Tidak ada            | Notifikasi dan alert  |
| Interaksi | Terbatas             | Kaya dan responsif    |

## Kesimpulan

Website tanpa JavaScript terlihat bagus secara visual (berkat CSS) tetapi tidak memberikan pengalaman interaktif yang diharapkan pengguna modern. JavaScript adalah kunci untuk membuat website yang benar-benar hidup dan responsif.
