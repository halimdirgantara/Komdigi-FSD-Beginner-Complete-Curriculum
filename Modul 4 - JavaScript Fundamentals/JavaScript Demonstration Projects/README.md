# JavaScript Demonstration Projects

## Overview

This folder contains two demonstration projects that showcase the difference between websites **without JavaScript** and **with JavaScript**.

## Projects

### 1. Static Website (No JavaScript)

**Folder**: `1. Static Website`

- Pure HTML and CSS only
- Demonstrates what a website looks like without JavaScript
- Limited interactivity
- No dynamic content or user interactions

### 2. Interactive Website (With JavaScript)

**Folder**: `2. Interactive Website`

- HTML, CSS, and JavaScript
- Demonstrates the power of JavaScript
- Dynamic content, animations, and user interactions
- Modern web experience

## How to Use

1. Open each project's `index.html` file in your browser
2. Compare the differences between the two websites
3. Notice how JavaScript enhances the user experience

## Key Differences

| Feature           | Static Website | Interactive Website  |
| ----------------- | -------------- | -------------------- |
| User Interactions | Limited        | Rich and responsive  |
| Dynamic Content   | None           | Real-time updates    |
| Animations        | CSS only       | JavaScript + CSS     |
| Form Validation   | None           | Real-time validation |
| Notifications     | None           | Toast notifications  |
| Loading States    | None           | Smooth transitions   |

## Learning Objectives

After exploring both projects, you should understand:

- Why JavaScript is essential for modern web development
- The difference between static and dynamic websites
- How JavaScript enhances user experience
- The power of client-side programming
