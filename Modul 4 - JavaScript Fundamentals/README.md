# Modul 4 - JavaScript Fundamentals

## Overview

Modul ini membahas dasar-dasar JavaScript untuk membuat halaman web yang interaktif. Siswa akan belajar tentang variabel, operator, tipe data, fungsi built-in, event dasar, dan DOM manipulation.

## Struktur Folder

- `1. Starter/` - Code awal untuk memulai praktik
- `2. Finished/` - Code final hasil praktik lengkap

## Learning Objectives

Setelah menyelesaikan modul ini, siswa akan mampu:

1. Memahami JavaScript sebagai bahasa scripting di browser
2. Menggunakan variabel, operator, dan tipe data
3. Menggunakan fungsi built-in (Math, String, Array)
4. Menangani event dasar (alert, prompt, confirm)
5. Melakukan DOM manipulation dasar
6. Membuat aplikasi penghitung umur yang interaktif

## Project Output

**Mini Project: Aplikasi Penghitung Umur**

- Form input untuk tanggal lahir
- Kalkulasi umur otomatis
- Validasi input
- Display hasil yang menarik
- Error handling
