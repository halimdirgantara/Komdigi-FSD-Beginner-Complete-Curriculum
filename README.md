# Komdigi Full Stack Development - Beginner Learning Repository

## Tentang Repository Ini

Repository ini berisi materi pembelajaran lengkap untuk memulai perjalanan menjadi Full Stack Developer. Disini kamu akan belajar dari nol sampai bisa membuat website interaktif yang lengkap dengan HTML, CSS, dan JavaScript.

Kurikulum ini dirancang khusus untuk pemula yang belum pernah coding sama sekali, atau yang baru mulai belajar web development. Setiap modul dibangun secara bertahap, mulai dari konsep paling dasar sampai project akhir yang mengintegrasikan semua yang sudah dipelajari.

## Struktur Pembelajaran

Repository ini terorganisir dalam modul-modul yang berurutan. Sangat disarankan untuk belajar secara berurutan dari Modul 2 sampai Final Project, karena setiap modul membangun dasar untuk modul berikutnya.

Setiap modul memiliki struktur yang sama:

- Folder Starter berisi kode awal untuk kamu mulai praktik
- Folder Finished berisi kode final yang bisa jadi referensi
- File README menjelaskan tujuan pembelajaran dan tugas yang harus dikerjakan

## Modul 2 - HTML Fundamentals

### Apa yang Dipelajari

Ini adalah titik awal perjalanan kamu. Modul ini mengajarkan dasar-dasar HTML, bahasa markup yang menjadi tulang punggung semua website.

Di modul ini kamu akan belajar:

- Struktur dasar dokumen HTML
- Elemen-elemen penting seperti heading, paragraf, list, dan link
- Cara membuat form untuk mengumpulkan data dari user
- Semantic HTML untuk membuat struktur yang bermakna
- Cara menambahkan gambar ke dalam halaman

### Project yang Dibuat

Kamu akan membuat halaman profil sederhana yang berisi:

- Informasi pribadi menggunakan heading dan paragraf
- Daftar skill dan pengalaman menggunakan list
- Form kontak dengan berbagai jenis input
- Gambar profil dan link ke sosial media

### Struktur Folder

- Starter: File HTML kosong dengan struktur dasar, tugas kamu adalah mengisi kontennya
- Finished: Contoh lengkap halaman profil yang sudah jadi

### Tips Belajar

Jangan terburu-buru. Pastikan kamu paham setiap elemen HTML sebelum lanjut ke modul berikutnya. Buka file Finished jika kamu stuck, tapi coba dulu sendiri dulu sebelum liat jawabannya.

## Modul 3 - CSS Fundamentals

### Apa yang Dipelajari

Setelah kamu bisa membuat struktur HTML, sekarang saatnya membuatnya terlihat menarik. CSS adalah bahasa styling yang mengontrol tampilan visual website.

Di modul ini kamu akan belajar:

- Selector untuk memilih elemen yang ingin di-style
- Box model untuk mengatur spacing dan ukuran elemen
- Typography untuk mengatur font, ukuran, dan style teks
- Warna dan background untuk membuat website lebih hidup
- Flexbox dan Grid untuk membuat layout yang rapi
- Media query untuk membuat website responsive di berbagai ukuran layar

### Project yang Dibuat

Kamu akan menghias halaman profil yang sudah dibuat di Modul 2 menjadi:

- Website yang menarik secara visual dengan warna dan typography yang konsisten
- Layout yang rapi menggunakan Flexbox atau Grid
- Website yang responsive, bisa tampil bagus di mobile, tablet, dan desktop

### Struktur Folder

- Starter: HTML dari Modul 2 dengan file CSS kosong, tugas kamu adalah menulis styling
- Finished: Halaman profil lengkap dengan styling modern dan responsive
- Pure HTML vs HTML with CSS Comparison: Perbandingan visual yang jelas betapa pentingnya CSS

### Tips Belajar

CSS bisa jadi tricky di awal karena ada banyak property. Jangan hafal semuanya, yang penting paham konsepnya. Pakai browser DevTools untuk experiment langsung tanpa save file berkali-kali.

## Modul 4 - JavaScript Fundamentals

### Apa yang Dipelajari

HTML dan CSS membuat website terlihat bagus, tapi belum interaktif. JavaScript lah yang membuat website bisa berinteraksi dengan user, melakukan kalkulasi, dan merespon aksi pengguna.

Di modul ini kamu akan belajar:

- Variabel untuk menyimpan data
- Tipe data seperti string, number, boolean, array, dan object
- Operator untuk manipulasi data
- Fungsi built-in JavaScript untuk berbagai operasi
- Event dasar seperti alert, prompt, dan confirm
- DOM manipulation dasar untuk mengubah konten halaman

### Project yang Dibuat

Kamu akan membuat aplikasi penghitung umur yang:

- Meminta input tanggal lahir dari user
- Menghitung umur secara otomatis
- Menampilkan hasil dengan format yang menarik
- Menangani error jika input tidak valid

### Struktur Folder

- Starter: HTML dan CSS dasar, tugas kamu adalah menambahkan JavaScript
- Finished: Aplikasi penghitung umur yang sudah berfungsi lengkap
- JavaScript Demonstration Projects: Dua contoh website yang menunjukkan perbedaan website statis tanpa JavaScript vs website interaktif dengan JavaScript

### Tips Belajar

JavaScript adalah bahasa programming pertama yang kamu pelajari. Awalnya mungkin confusing, tapi normal. Practice is key. Coba ubah-ubah kode di console browser untuk melihat hasilnya langsung.

## Modul 5 - DOM Manipulation & Interactivity

### Apa yang Dipelajari

Modul ini memperdalam kemampuan JavaScript kamu, khususnya dalam hal berinteraksi dengan elemen HTML dan membuat website yang benar-benar interaktif dan responsif terhadap input user.

Di modul ini kamu akan belajar:

- Cara mengakses elemen HTML dari JavaScript
- Cara mengubah konten, style, dan atribut elemen secara dinamis
- Event listeners untuk mendeteksi klik, input, submit, dan event lainnya
- Form validation untuk memastikan data yang dimasukkan user valid
- Local storage untuk menyimpan data di browser
- Debugging menggunakan browser console dan DevTools

### Project yang Dibuat

Kamu akan membuat aplikasi to-do list yang fungsional:

- Menambah dan menghapus task
- Menandai task sebagai selesai
- Filter task berdasarkan status (semua, aktif, selesai)
- Menyimpan data di local storage sehingga tidak hilang saat refresh
- Desain yang responsive dan smooth animations

### Struktur Folder

- Starter: Template HTML dan CSS dasar untuk to-do list, tugas kamu adalah menambahkan functionality dengan JavaScript
- Finished: To-do list aplikasi yang lengkap dengan semua fitur
- JavaScript Demonstration Projects: Perbandingan to-do list statis tanpa interaktivitas vs to-do list interaktif dengan DOM manipulation

### Tips Belajar

Ini adalah modul yang paling banyak praktiknya. Jangan skip project-nya karena ini yang akan mengasah skill kamu. Jika stuck, baca error message di console, biasanya sudah jelas apa masalahnya.

## Final Project - Interactive Personal Profile Website

### Apa yang Dipelajari

Final project ini adalah ujian akhir yang mengintegrasikan semua yang sudah kamu pelajari dari Modul 2 sampai Modul 5. Disini kamu akan membuat website portfolio pribadi yang lengkap, interaktif, dan professional.

Website yang kamu buat akan memiliki:

- Struktur HTML yang semantic dan accessible
- Styling CSS modern dengan animations dan transitions
- Dark mode toggle untuk tema gelap dan terang
- Mobile navigation dengan hamburger menu
- Form validation yang comprehensive
- Portfolio filtering berdasarkan kategori
- Smooth scrolling dan scroll animations
- Responsive design untuk semua device

### Struktur Folder

- Starter: Template HTML dasar dengan struktur sections sudah ada, file CSS dan JavaScript kosong yang harus kamu isi
- Finished: Website portfolio lengkap dengan semua fitur (jangan push folder ini, ini hanya referensi)

### Tips Belajar

Ini adalah project terbesar kamu. Jangan terburu-buru dan jangan panik. Break down menjadi bagian-bagian kecil. Selesaikan satu fitur dulu baru lanjut ke fitur berikutnya. Website ini bisa kamu gunakan sebagai portfolio untuk apply kerja nanti.

## Cara Menggunakan Repository Ini

### Untuk Pemula Total

Jika kamu benar-benar baru di dunia coding:

1. Mulai dari Modul 2 - HTML Fundamentals
2. Baca README di setiap modul untuk memahami tujuan pembelajaran
3. Buka folder Starter dan mulai coding sesuai instruksi
4. Jika stuck lebih dari 30 menit, baru lihat folder Finished sebagai referensi
5. Jangan copy-paste langsung, coba pahami dulu kenapa kodenya seperti itu
6. Practice dengan mengubah-ubah kode untuk melihat hasilnya

### Untuk yang Sudah Ada Basic

Jika kamu sudah pernah lihat HTML, CSS, atau JavaScript sebelumnya:

1. Quick review Modul 2 dan 3 untuk memastikan foundation kamu solid
2. Fokus lebih banyak di Modul 4 dan 5 karena ini yang biasanya lebih challenging
3. Challenge yourself dengan menambahkan fitur tambahan di setiap project
4. Gunakan Final Project sebagai showcase skill kamu

### Tips Umum

- Setup text editor yang nyaman. Visual Studio Code adalah pilihan populer dan gratis
- Install extension HTML CSS Support untuk autocomplete
- Gunakan browser modern seperti Chrome atau Firefox dengan DevTools yang bagus
- Jangan lupa save file dan refresh browser setiap kali ada perubahan
- Git commit setiap selesai modul untuk track progress kamu
- Gabung komunitas programming untuk bertanya jika stuck

## Troubleshooting Umum

File HTML tidak muncul di browser: Pastikan file disimpan dengan ekstensi .html dan dibuka dengan browser, bukan text editor.

CSS tidak bekerja: Cek apakah file CSS sudah di-link di HTML dengan benar. Cek juga path file-nya.

JavaScript error di console: Baca pesan error-nya, biasanya sudah jelas masalahnya. Pastikan variabel sudah didefinisikan sebelum dipakai.

Website tidak responsive: Pastikan sudah ada meta viewport di HTML dan menggunakan media query di CSS.

Local storage tidak bekerja: Pastikan membuka website via file:// atau localhost, bukan drag file langsung ke browser.
