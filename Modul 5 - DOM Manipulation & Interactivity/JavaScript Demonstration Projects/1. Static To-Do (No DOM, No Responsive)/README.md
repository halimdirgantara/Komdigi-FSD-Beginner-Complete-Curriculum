## 1. Static To-Do (No DOM, No Responsive)

Proyek ini menunjukkan bagaimana tampilan To-Do List tanpa interaktivitas dan tanpa responsive design.

Keterbatasan yang terlihat:

- Tidak bisa menambah, toggle, atau menghapus item (tidak ada DOM manipulation / event listeners)
- Angka statistik tidak berubah dinamis
- Layout fixed width, kurang nyaman di layar kecil

Cara menjalankan:

- Buka `index.html` langsung di browser.

Relasi ke materi Modul 5:

- Kontras dengan bagian DOM (akses, modifikasi elemen) dan Event Listeners yang tidak digunakan di sini.
