## Demonstration Projects (Modul 5)

Berikut dua proyek perbandingan untuk menunjukkan dampak penggunaan DOM Manipulation dan Responsive Design.

1. 1. Static To-Do (No DOM, No Responsive)

- Fokus pada HTML statis, tanpa JavaScript DOM events dan tanpa responsive layout.
- Tujuan: Memperlihatkan keterbatasan tanpa interaktivitas dan tanpa adaptasi layar.

2. 2. Interactive To-Do (DOM + Responsive)

- Menggunakan DOM manipulation, event listeners, dan localStorage sesuai materi modul.
- Menggunakan responsive CSS (media query, fluid layout) agar nyaman di berbagai ukuran layar.

Lihat masing-masing folder untuk instruksi dan cara menjalankan.
