## 2. Interactive To-Do (DOM + Responsive)

Proyek ini menerapkan materi Modul 5:

- DOM Manipulation: createElement, classList, textContent/innerHTML, remove
- Event Listeners: submit, click, input
- Local Storage: persist data antar refresh
- Responsive Design: grid layout, media queries

Cara menjalankan:

- Buka `index.html` di browser.
- Tambahkan tugas, toggle selesai, hapus, dan coba filter.
- Refresh halaman untuk melihat data tetap tersimpan (localStorage).

Perbedaan dibanding proyek statis:

- Interaksi real-time melalui event listeners
- UI diperbarui dinamis via DOM manipulation
- Statistik otomatis
- Tampilan nyaman di layar kecil berkat media query
