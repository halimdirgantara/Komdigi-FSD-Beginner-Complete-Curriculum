# Modul 5 - DOM Manipulation & Interactivity

## Overview

Modul ini membahas DOM manipulation dan interaktivitas dalam JavaScript. Siswa akan belajar tentang mengakses dan memodifikasi elemen DOM, event listeners, form validation, dan debugging.

## Struktur Folder

- `1. Starter/` - Code awal untuk memulai praktik
- `2. Finished/` - Code final hasil praktik lengkap

## Learning Objectives

Setelah menyelesaikan modul ini, siswa akan mampu:

1. Mengakses dan memodifikasi elemen DOM
2. Menggunakan event listeners (click, input, submit)
3. Membuat form validation sederhana
4. Melakukan debugging dengan Inspect & console
5. Membuat to-do list interaktif yang fungsional

## Project Output

**Mini Project: To-do List Interaktif**

- Add/remove tasks
- Mark tasks as complete
- Filter tasks (all, active, completed)
- Local storage untuk persist data
- Responsive design
- Smooth animations
