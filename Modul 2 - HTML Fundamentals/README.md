# Modul 2 - HTML Fundamentals

## Overview

Modul ini membahas dasar-dasar HTML untuk membangun struktur halaman web. Siswa akan belajar tentang elemen-elemen HTML dasar, struktur dokumen, dan cara membuat halaman profil sederhana.

## Struktur Folder

- `1. Starter/` - Code awal untuk memulai praktik
- `2. Finished/` - Code final hasil praktik lengkap

## Learning Objectives

Setelah menyelesaikan modul ini, siswa akan mampu:

1. Memahami struktur dasar dokumen HTML
2. Menggunakan elemen teks, heading, list, link, dan image
3. Membuat form dan input elements
4. Menerapkan semantic HTML
5. Membuat halaman profil sederhana

## Project Output

**Mini Project: Halaman Profil Sederhana**

- Struktur HTML lengkap dengan semantic elements
- Informasi pribadi dengan heading dan paragraf
- Daftar skill dan pengalaman
- Form kontak dengan berbagai input types
- Gambar profil dan link sosial media
